# CogniNautas

**¡Bienvenido a CogniNautas, tu aplicación de minijuegos para fortalecer la memoria de trabajo!**

## ¿Qué es CogniNautas?

CogniNautas es una aplicación interactiva diseñada especialmente para niños de 8 a 12 años. Su objetivo es ayudarte a descubrir cómo funciona tu memoria de trabajo y fortalecerla a través de divertidos minijuegos basados en la ciencia cognitiva.

## ¿Qué es la memoria de trabajo?

La memoria de trabajo es como el bloc de notas de tu cerebro, te ayuda a recordar y usar información por cortos periodos, como seguir instrucciones, recordar números o resolver problemas.

- [Harvard Medical School: Working Memory](https://www.health.harvard.edu/newsletter_article/working-memory)
- [Child Mind Institute: What is Working Memory?](https://childmind.org/article/what-is-working-memory/)
- [APA: Memory and Learning in Children](https://www.apa.org/topics/learning-memory/children)

## ¿Por qué usar CogniNautas?

- **Descubrir:** Aprende sobre tu memoria de trabajo.
- **Jugar:** Ponla a prueba con minijuegos entretenidos.
- **Mejorar:** Aplica estrategias que te ayudarán a fortalecerla.

## ¿Cómo usar la aplicación?

1. **Elige un minijuego:** Encuentra diferentes juegos que ejercitan tu memoria de trabajo.
2. **Lee las instrucciones:** Antes de jugar, revisa cómo funciona el minijuego.
3. **¡Diviértete jugando!** Cada intento es una oportunidad para mejorar.
4. **Revisa tus resultados:** Comprueba tu progreso y aprende más sobre tu memoria.

## Consejos para sacarle el máximo provecho

- Juega en un lugar tranquilo.
- Lee con atención las instrucciones.
- Practica varias veces... ¡verás cómo avanzas!
