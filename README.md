# ¡Bienvenido a CogniNautas!

Hola explorador,

¡Nos alegra que estés aquí para comenzar una aventura mental! CogniNautas es una aplicación diseñada especialmente para niños como tú, entre 8 y 12 años, para ayudarte a descubrir, jugar y fortalecer tu memoria de trabajo. 

## ¿Qué es la memoria de trabajo?

La memoria de trabajo nos ayuda a recordar y usar información por cortos periodos de tiempo, por ejemplo: recordar los números de teléfono, seguir instrucciones, o resolver problemas matemáticos. ¡Es como el bloc de notas de tu cerebro!

- **Fuente:** [Centro de aprendizaje y memoria de trabajo: Harvard Medical School](https://www.health.harvard.edu/newsletter_article/working-memory)
- **Fuente:** [Memoria de trabajo en niños: Child Mind Institute](https://childmind.org/article/what-is-working-memory/)

## ¿Cuál es el objetivo de esta aplicación?

CogniNautas quiere ayudarte a:

- Descubrir cómo funciona tu memoria de trabajo.
- Jugar divertidos minijuegos que la ponen a prueba.
- Aprender cómo mejorar tu memoria de trabajo usando estrategias divertidas.

## ¿Cómo se utiliza CogniNautas?

1. **Elige un minijuego:** En la pantalla principal encontrarás diferentes minijuegos, cada uno diseñado para ejercitar tu memoria de trabajo.
2. **Sigue las instrucciones:** Antes de empezar cada minijuego, tendrás una breve explicación sobre cómo jugar.
3. **Juega y diviértete:** Realiza las actividades, ¡no te preocupes si te equivocas, cada intento cuenta!
4. **Revisa tus resultados:** Después de jugar, podrás ver cómo te fue, ¡y aprenderás más sobre tu memoria!

## Consejos para aprovechar al máximo la aplicación

- Busca un lugar tranquilo para jugar.
- Lee con atención las instrucciones de cada minijuego.
- ¡Intenta varias veces y verás cómo mejoras!

---

**Recuerda:** La memoria de trabajo es muy importante para el aprendizaje y la vida diaria, y con CogniNautas podrás mejorarla mientras te diviertes.

¡Prepárate para embarcarte en una misión mental!
